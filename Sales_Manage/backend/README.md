# Sales Management System

A full-stack **Sales Management System** built with **React** (frontend) and **Node.js + Express + MongoDB** (backend).  
The system allows managing and analyzing sales data with features like search, filtering, sorting, pagination, and sales statistics.

## Features

- Display sales data in a tabular format.  
- Search by customer name or phone number.  
- Filter sales by:
  - Customer Region  
  - Gender  
  - Product Category  
  - Payment Method  
  - Date Range  
- Sort by Date, Quantity, or Customer Name.  
- Pagination support for large datasets.  
- Statistics panel showing:
  - Total Units Sold  
  - Total Amount  
  - Total Discount  

---

## Tech Stack

- **Frontend:** React, Tailwind CSS  
- **Backend:** Node.js, Express  
- **Database:** MongoDB (with Mongoose ODM)  
- **Others:** CORS, dotenv
